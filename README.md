# ACA-TechGen-CSharp

This repository contains all homework assignments and projects for the **Armenian Code Academy TechGen C# Backend Developer** course.

---

## Course Modules

This course covers the following modules:

- **.NET & C# Fundamentals**
- **OOP — Classes, Inheritance, Interfaces**
- **Delegates, Events & Patterns**
- **Design Patterns**
- **LINQ**
- **Multithreading**
- **Databases & ORM (Entity Framework Core)**
- **REST APIs (ASP.NET Core)**
- **Docker & Deployment**

---

## Repository Structure & Workflow

- Each assignment or module is contained in its own solution.
- Each new homework or module is developed on a separate Git branch.
- After completing an assignment, a pull request (PR) is created for code review.
- Upon approval, the task branch is merged into `master`.
